# Analisis Data Toko Buku (SQL & Python)

Proyek ini bertujuan untuk menganalisis performa operasional toko buku dan melakukan segmentasi pelanggan untuk strategi pemasaran yang lebih efektif.

## Fitur Utama:
- **Ekstraksi Data:** Menggunakan SQL Joins dan Agregasi untuk menarik data dari PostgreSQL.
- **Analisis Tren:** Menghitung rata-rata bergerak (Moving Average) dengan teknik *resampling* untuk menangani data transaksi yang tidak berurut.
- **Segmentasi Pelanggan (RFM):** Mengelompokkan pelanggan menjadi kategori 'VIP', 'Sultan Beresiko', dll, menggunakan logika Pandas.
- **Visualisasi:** Grafik batang dan garis menggunakan Matplotlib untuk memudahkan pengambilan keputusan bisnis.

## Tech Stack:
- Database: PostgreSQL
- Language: Python
- Libraries: Pandas, SQLAlchemy, Matplotlib
